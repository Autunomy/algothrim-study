题意：
给一个n的排列a，给a排序，规则是i < j 且j-i == k的元素可以互相交换，求k的最大值

思路：
将每个数与其应该在的位置之差(如果差为0就不保存)保存起来，k的值就是这些差值的gcd
例如
1. 一个数与其应在的位置之差为6另一个为3，则k应该为3
2. 一个数与其应在的位置之差为3另一个为2，则k应该为1

                if(res == -1) {
                    res = temp;
                    continue;
                }

                res = Math.min(res,(int)gcd(res,temp));
            }
        }

    }
