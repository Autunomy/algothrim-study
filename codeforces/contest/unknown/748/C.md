题意:
给出一个行走方式，这个行走方式会将集合中的点全部都遍历一遍(可能会收尾相连,集合中可能包含重复点)，求集合中点最少的数量


思路:
每次行走的时候都判断一下与上一个点之间是否有反向的路径，加上上一个点到当前位置的路径中只有L和U，这一步却向R或D走，则说明当前两个路径中间至少有一个点

        int n = sc.nextInt();
        String s = sc.next();
        int res = 1;

        Set<Character> set = new HashSet<>();

        for(int i=0;i<n;++i){
            char c = s.charAt(i);
            if(c == 'U'){
                if(set.contains('D')){
                    res++;
                    set.clear();
                }
                set.add(c);
            }else if(c == 'D'){
                if(set.contains('U')){
                    res++;
                    set.clear();
                }
                set.add(c);
            }else if(c == 'L'){
                if(set.contains('R')){
                    res++;
                    set.clear();
                }
                set.add(c);
            }else{
                if(set.contains('L')){
                    res++;
                    set.clear();
                }
                set.add(c);
            }
        }

        System.out.println(res);

    }
