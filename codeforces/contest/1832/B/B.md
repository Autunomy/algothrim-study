# B

题意：给定一个数组，进行k次操作
1. 删除数组中最小的两个
2. 删除数组中最大的一个

求操作完成后数组的最大和

思路：模拟以下即可，因为题目给定n的总和不会超过2e5

public class Main {

    void solve(){
        int n = sc.nextInt();
        int k = sc.nextInt();

        int[] arr = scIntArr(n);
        Arrays.sort(arr);

        long[] pre = new long[n+1];
        for(int i=1;i<=n;++i){
            pre[i] = pre[i-1] + arr[i-1];
        }
        long res = 0L;
        int i=2*k,j=n;
        k++;
        while(k -- > 0){
            res = Math.max(res,pre[j]-pre[i]);
            j--;
            i-=2;
        }
        System.out.println(res);
    }
